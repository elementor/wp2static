Static site generator functionality for WordPress.

== Description ==

WP2Static's core plugin provides static site export functionality.

Add-ons are available to provide advanced functionality.

See our [**homepage**](https://wp2static.com) for documentation, support and add-ons.



== Installation ==

1. Go to Plugins > Add New
2. Search for "WP2Static"
3. Click on the Install Now button
4. Activate the plugin and find it under the Tools menu
